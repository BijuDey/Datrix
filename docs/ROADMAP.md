# Product Roadmap

This roadmap outlines planned evolution areas for Datrix.

## Near Term

- Add complete "Create New Organization" flow (see FEATURE_NEW_ORGANIZATION_FLOW.md)
- Improve SQL explorer UX for large result sets
- Expand logs filtering and export capabilities
- Harden role-based access checks in collaboration flows

## Mid Term

- Introduce connection adapter abstraction for additional engines
- Add query execution safeguards (timeouts, row limits, cancellation)
- Add richer health diagnostics for connectors

## Long Term

- MongoDB support (see FEATURE_MONGODB_SUPPORT.md)
- Plugin/extension model for third-party connectors
- Advanced observability dashboards

## Notes

Roadmap items are subject to prioritization and contributor availability.
